### 2. Return on Investment (ROI)
```
ROI = ((Revenue - Cost) / Cost) Ã— 100%

Contoh:
Revenue = Rp 50.000
Cost = Rp 10.000
ROI = ((50.000 - 10.000) / 10.000) Ã— 100%
    = 400%

Artinya: Setiap Rp 1 investasi â†’ Rp 4 profit
```

### 3. Break-Even Point (BEP)
