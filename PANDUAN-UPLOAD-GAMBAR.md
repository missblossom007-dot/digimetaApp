# Panduan Upload Gambar Cover Buku

## Langkah-langkah:

### 1. Download Cover Buku Asli
Cari dan download cover buku dari sumber berikut:
- Google Images (cari: "atomic habits book cover high quality")
- Amazon.com (klik kanan pada gambar buku → Save image)
- Goodreads.com
- Situs resmi penerbit

### 2. Nama File yang Dibutuhkan
Simpan gambar dengan nama berikut di folder `public/book-covers/`:

- `atomic-habits.jpg` atau `atomic-habits.png`
- `rich-dad-poor-dad.jpg` atau `rich-dad-poor-dad.png`
- `psychology-of-money.jpg` atau `psychology-of-money.png`
- `deep-work.jpg` atau `deep-work.png`
- `warren-buffett-way.jpg` atau `warren-buffett-way.png`

### 3. Lokasi Folder
```
digimetashop-nextjs/
└── public/
    └── book-covers/
        ├── atomic-habits.jpg
        ├── rich-dad-poor-dad.jpg
        ├── psychology-of-money.jpg
        ├── deep-work.jpg
        └── warren-buffett-way.jpg
```

### 4. Rekomendasi Ukuran Gambar
- Lebar: 400-600px
- Tinggi: 600-900px
- Format: JPG atau PNG
- Ukuran file: < 500KB per gambar

### 5. Setelah Upload
Setelah Anda memasukkan gambar ke folder `public/book-covers/`, 
beritahu saya dan saya akan update kode untuk menggunakan gambar tersebut.
