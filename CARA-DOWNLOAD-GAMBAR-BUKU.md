# Cara Download Gambar Cover Buku

Gambar buku Warren Buffett Way, Psychology of Money, dan Deep Work belum muncul karena perlu didownload manual.

## Langkah Download:

### 1. Buka Browser dan Download Gambar:

**Psychology of Money:**
1. Buka: https://www.google.com/search?q=psychology+of+money+book+cover&tbm=isch
2. Pilih gambar cover buku yang jelas
3. Klik kanan → Save Image As
4. Simpan sebagai: `psychology-of-money.jpg`

**Deep Work:**
1. Buka: https://www.google.com/search?q=deep+work+cal+newport+book+cover&tbm=isch
2. Pilih gambar cover buku yang jelas
3. Klik kanan → Save Image As
4. Simpan sebagai: `deep-work.jpg`

**Warren Buffett Way:**
1. Buka: https://www.google.com/search?q=warren+buffett+way+book+cover&tbm=isch
2. Pilih gambar cover buku yang jelas
3. Klik kanan → Save Image As
4. Simpan sebagai: `warren-buffett-way.jpg`

### 2. Simpan ke Folder:

Pindahkan semua file ke folder:
```
D:\Downloads\digimetashop-nextjs\public\book-covers\
```

### 3. Restart Server:

Setelah gambar tersimpan, restart development server:
- Tekan `Ctrl+C` di terminal
- Jalankan lagi: `npm run dev`

### 4. Refresh Browser:

Tekan `Ctrl+Shift+R` untuk hard refresh
